# Vivaldi UI Tweaks

![License](https://img.shields.io/badge/License-MIT-blue.svg)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

Minimal custom UI modifications for Vivaldi browser. Kiosk-style full screen mode with optional floating tabs.

> **Note:** This project is no longer maintained.

## Features

- Full screen kiosk mode by default
- Hidden toolbars and menus
- Optional floating tabs (bottom-right corner)
- Modular CSS structure

## Installation

1. Go to `vivaldi://experiments/`
2. Enable "Allow for using CSS modifications"
3. Open Settings → Appearance → Custom UI Modifications
4. Select folder containing these CSS files
